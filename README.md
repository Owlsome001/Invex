# InvEx

InvEx(Inventory Excellence) is an Inventory Management System. Its avails some features to help a warehouse staff managing stocks and tracking some metrics about the whole warehouse.




## Features

- Add stocks
- Add mouvement
- Approve mouvement
- Update mouvement
- StockSheet in Excel
